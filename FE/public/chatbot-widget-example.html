Betabase Chatbot Widget Integration Guide

Basic Integration

Add this script tag to your HTML before the closing </body> tag:

<script src="https://your-domain.com/chatbot-widget.js"></script>

With Custom Configuration

You can customize the chatbot by setting a configuration object before loading the script:

<script> window.ChatbotConfig = { apiUrl: 'https://api.betabase.pro/api/v1', adminEmail: 'admin@betabase.pro', botAvatar: 'https://your-domain.com/bot.jpg' }; </script> <script src="https://your-domain.com/chatbot-widget.js"></script>

Configuration Options

  • apiUrl (string): The API endpoint URL. Default: 'https://api.betabase.pro/api/v1'
  • adminEmail (string): Admin email for receiving transcripts. Default: 'admin@betabase.pro'
  • botAvatar (string): URL to the bot avatar image. Default: Uses default avatar

Complete Example

<!DOCTYPE html> <html lang="en"> <head> <meta charset="UTF-8"> <meta name="viewport" content="width=device-width, initial-scale=1.0"> <title>My Website</title> </head> <body> <h1>Welcome to My Website</h1> <p>Your content here...</p> <!-- Chatbot Widget --> <script> window.ChatbotConfig = { apiUrl: 'https://api.betabase.pro/api/v1', adminEmail: 'admin@betabase.pro', botAvatar: 'https://your-domain.com/bot.jpg' }; </script> <script src="https://your-domain.com/chatbot-widget.js"></script> </body> </html>

Notes

  • The widget automatically loads React from CDN if not already present
  • The chatbot will appear as a floating button in the bottom-right corner
  • All functionality from the original chatbot is preserved
  • The widget is self-contained and won't interfere with your existing website
  • Make sure your API endpoint allows CORS requests from your domain
